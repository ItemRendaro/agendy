# Vuepressを使ってGithub Pagesにデプロイする際の設定について

2024-02-10 18:20

以下の設定は、Visual Studio Codeのコンテキストメニューをカスタマイズする例です。この設定は、settings.jsonファイルに追加されます。

"editor/context": [
    {
        "command": "bookmarks.toggle",
        "group": "bookmarks",
        "when": "editorTextFocus && config.bookmarks.showCommandsInContextMenu"
    },
    {
        "command": "bookmarks.jumpToNext",
        "group": "bookmarks@1",
        "when": "editorTextFocus && config.bookmarks.showCommandsInContextMenu"
    },
    {
        "command": "bookmarks.jumpToPrevious",
        "group": "bookmarks@1",
        "when": "editorTextFocus && config.bookmarks.showCommandsInContextMenu"
    }
]

これは、右クリックしたときに表示されるコンテキストメニューに新しい項目を追加します。 それぞれの項目は、editor/context内のコマンドを定義しています。 この例では、"bookmarks.toggle"、"bookmarks.jumpToNext"、"bookmarks.jumpToPrevious"という3つのコマンドが指定されています。

これらの設定は、コマンドが実際に拡張機能によって定義されている必要がないことを示しています。 このようにして、既存のコマンドをメニューに追加したり、カスタムコマンドを作成してメニューに追加したりすることができます。

この設定により、コンテキストメニューがカスタマイズされ、特定の操作がより簡単に実行できるようになります。
